# Photo Snap - 自动化拍照技能

## 快速使用

**只需说：**
- "拍照"
- "截图"
- "拍张照"
- "截个图"

**小梦会自动：**
1. 📷 打开 Photo Booth
2. 📸 截取屏幕截图
3. ☁️ 上传到飞书云盘
4. 💬 发送云盘链接给你

## 技能位置

`~/clawd/skills/photo-snap/SKILL.md`

## 权限要求

- ✅ macOS 屏幕录制权限（已授权）
- ✅ 飞书文档创建权限
- ✅ 飞书消息发送权限

## 输出说明

- **截图保存：** `~/Desktop/screenshot_YYYYMMDD_HHMMSS.png`
- **云盘链接：** 飞书文档格式，可随时查看下载
- **处理时间：** 约 5-10 秒
