# 知识库同步指南

## 🔄 如何让 OpenClaw 读取知识库

### 方法 1：直接读取（推荐日常使用）

随时对我说：
- "读取 knowledge/01_investments/trades/ 的最新文件"
- "帮我总结本周的市场笔记"
- "搜索关于 XX 策略的笔记"

### 方法 2：自动同步到 MEMORY.md

运行以下命令让小梦索引知识库：

```bash
# 让小梦读取并索引知识库
openclaw agent read ~/clawd/knowledge/INDEX.md
```

### 方法 3：定期自动同步

可以在 HEARTBEAT.md 中配置定期任务：

```markdown
# HEARTBEAT.md
## 每日任务
- [ ] 读取今日市场笔记
- [ ] 检查待办事项
- [ ] 同步重要发现到 MEMORY.md
```

## 📋 最佳实践

### 每日流程
1. **开盘前**：查看昨日笔记和今日计划
2. **收盘后**：在 daily_notes/ 创建今日市场笔记
3. **有交易时**：在 trades/ 创建交易记录
4. **周末**：整理本周笔记，归档到周报

### 命名规范
- 日期文件：`YYYY-MM-DD.md`
- 交易记录：`TRADE_YYYY-MM-DD_品种.md`
- 会议纪要：`MEETING_YYYY-MM-DD_主题.md`

### 标签使用
- `#交易` - 所有交易相关
- `#宏观` - 宏观分析
- `#策略` - 策略研究
- `#产品` - 产品运作
- `#重要` - 关键信息
- `#待办` - 待办事项

---

## 🔍 快速检索

小梦支持以下查询方式：

1. **按时间**："帮我找 3 月份的市场笔记"
2. **按标签**："搜索所有带 #策略 的文件"
3. **按内容**："找一下关于国债收益率的分析"
4. **按类型**："列出最近的会议纪要"
